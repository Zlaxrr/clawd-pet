Quick start

  1. Unzip the folder anywhere you like.
  2. Double-click start-clawd.vbs. The first run downloads the official Clawd sprites from claude.ai (needs internet, a few seconds).
  3. Clawd walks onto your taskbar. To close him: right-click Clawd → Bye Clawd, or run stop-clawd.vbs.
  4. Optional: right-click tools\claude-watch-on.ps1 → Run with PowerShell to let him show what Claude Code is doing.

Needs Windows 10 or 11 (PowerShell 5.1 is already there) and internet the first time. If Windows SmartScreen warns about the .vbs, click More info → Run anyway — it's a generic "this is a script" flag, not anything specific to this project.

Clawd & his artwork belong to Anthropic; they're downloaded from claude.ai on first run, never bundled. Fan project, unofficial. Code is MIT.
